# hex to rgb()

**hex to rgb()** is a simple utility web app that converts hexadecimal CSS colour values to CSS *rgb()* function calls.

## Using the web app

Go to (https://bjoernkw.github.io/hexrgb/).

## Built With

* [Bootstrap](https://getbootstrap.com)
* JavaScript
